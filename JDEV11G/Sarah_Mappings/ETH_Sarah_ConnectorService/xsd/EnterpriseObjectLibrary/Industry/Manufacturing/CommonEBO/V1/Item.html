Item

EBO Type Enterprise Business Object
CCTS Type
CCTS Dictionary Entry Name Item. Details
Description A product is a good or service which can be bought and sold. A product is anything that can be offered to a market that might satisfy a want or need. Item EBO contains the definition of a sellable product or service or a manufactured or procured inventory item. It describes the form, fit and function of the item. It contains attributes that comprise the item definition in the master organization as well as inventory organizations. It is synonymous with Product from a CRM application standpoint. It provides a way to identify the item in any other system such as that of the manufacturer or supplier or customer or standard such as GTIN

ItemRepetitivePlanning/AcceptableRateIncreasePercent

CCTS Dictionary Entry Name Item Repetitive Planning. Acceptable Rate Increase. Percent
Description This field represents the percentage of the current aggregate rate by which your suggested daily rates can vary from current aggregate rates within a repetitive planning period inside the planning time fence. The repetitive planning process calculates acceptable rate increase by multiplying the current aggregate rate by the acceptable rate increase percent.
