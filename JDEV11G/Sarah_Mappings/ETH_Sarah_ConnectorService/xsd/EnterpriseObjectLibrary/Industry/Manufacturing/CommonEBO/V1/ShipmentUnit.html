ContainerItemInstanceRange Container Item. Container Item Instance Range. Container Item Instance Range
A range of serial numbers, included in the container. The serial number may be specified using multiple occurrence of ContainerItemInstanceRanges (please note that multiple sets are only used when the serial number ranges are not continuous) or multiple occurrence of ContainerItemInstances or a combination of both. ContainerItemInstanceRange should not be used to specify an individual serial number, for example using the same starting and ending serial number or filling the starting serial number but leaving the ending serial number blank. This is ambiguous since a missing or empty element (tag) could be interpreted differently by various groups.
