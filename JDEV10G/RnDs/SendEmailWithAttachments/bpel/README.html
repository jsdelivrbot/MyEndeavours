SendEmailWithAttachments

Abstract

The SendEmailWithAttachments process is used to send an email with attachment.

How to Run

Run obant.bat from the 130.SendEmailWithAttachments directory to deploy the process to the BPEL engine. When the process is deployed, test by doing the following:

  1. Login to the BPEL console for your server, select the SendEmailWithAttachments process, and go to the 'initiate' tab.
  2. The process has a default message that can be used to initiate the process. The default message sets the email TO to jcooper and the file samplecover.pdf as the email. The attachmentURI can be one of the following
    1. As a file URI. For example, 'file:C:/documents/samplecover.pdf'
    2. As an http URI. For example, 'http://host1/documents/samplecover.pdf'
    3. Relative to the BPEL suitcase. For example, 'samplecover.pdf' represents the file samplecover.pdf that is deployed with the BPEL suitcase at the same level as the bpel.xml
  3. Check the email account of jcooper, there should be an email with the attachment.
